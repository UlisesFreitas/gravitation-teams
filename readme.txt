A Plugin to integrate teams

== Description ==
Gravitation teams, is a plugin to display teams on your site, with ShortCodes

== Installation ==
Download and install from WordPress.org.
Configure as you see fit, under Gravitation / Gravitation teams.
1. Upload `gravitation-teams.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [gravitaiton_teams] in your templates
